# 🛒 Shopping Cart Demo

A simple and colorful shopping cart web application built using **HTML, CSS, and JavaScript**.  
This project demonstrates core front‑end skills such as DOM manipulation, event handling, responsive design, and local storage.

---

## ✨ Features
- Add items (Apple, Banana, Orange) to the cart
- Display cart items with **quantities and prices in ₹ (Rupees)**
- Update total dynamically
- Remove items from the cart
- Checkout summary with item count and total amount
- Responsive and colorful UI with hover effects
- Clean separation of HTML, CSS, and JavaScript
## 📂 Project Structure
shopping-cart/
│── index.html   # Main HTML file
│── style.css    # Styling (colorful, responsive design)
│── script.js    # Cart logic (add, remove, checkout)
